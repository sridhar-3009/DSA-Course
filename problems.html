1. Two Sum
Easy Array HashMap

Given an array of integers and a target, return indices of two numbers that add up to target. Each input has exactly one solution.

Brute Force O(n²)

Nested loops — try every pair (i,j). Simple but slow.

Optimal O(n)

HashMap: store seen values. For each element, check if complement exists in map.

Brute Force O(n²)

For every starting index, extend until duplicate found. O(n²) substrings.

Sliding Window O(n)

5. Climbing Stairs
Easy DP Recursion

You can climb 1 or 2 steps at a time. How many distinct ways can you reach the top of n stairs? The answer is the Fibonacci sequence.

Brute Force O(2ⁿ)

Naive recursion — exponentially redundant calls. Watch the call counter explode.

DP Table O(n)

Fill dp[0..n] bottom-up. dp[i] = dp[i-1] + dp[i-2].

Space-Opt O(n) O(1)

7. Merge Intervals
Medium Array Sorting

Given a list of intervals, merge all overlapping intervals. Return the minimal non-overlapping list.

Brute Force O(n²)

Repeatedly scan for any overlapping pair and merge. Until none remain.

Sort + Scan O(n log n)

Sort by start time. Single linear scan — merge adjacent overlaps.

8. Number of Islands
Medium Graph DFS BFS

Given a 2D binary grid of '1's (land) and '0's (water), count the number of islands. An island is surrounded by water and formed by connecting adjacent lands horizontally or vertically.

DFS O(m×n)

For each unvisited land cell, DFS recursively marks all connected land. Count roots.

BFS O(m×n)

Same complexity — use a queue instead of recursion. Avoids stack overflow on large grids.
