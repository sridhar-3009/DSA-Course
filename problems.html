P13 — Easy Binary Search
ArraysDivide & Conquer

Search for a target in a sorted array. Maintain left (L) and right (R) pointers. At each step, compute mid = (L+R)//2 and compare. If target < mid, search left half; if target > mid, search right half; else found. Each step eliminates half the search space — log₂(n) steps maximum.

Linear search: O(n) · O(1) Binary search: O(log n) · O(1)
◆Each comparison eliminates half the remaining candidates. After k steps, n/2ᵏ elements remain. Solving n/2ᵏ=1 gives k=log₂n — that's why binary search is O(log n).
